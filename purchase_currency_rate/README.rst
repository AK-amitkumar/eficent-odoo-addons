==============================
Account Purchase Currency Rate
==============================

This module adds the field currency_rate on purchase order in order to force
 the currency rate to be used.

Usage
=====

.. image:: https://odoo-community.org/website/image/ir.attachment/5784_f2813bd/datas
   :alt: Try me on Runbot
   :target: https://runbot.odoo-community.org/runbot/95/9.0

Known issues / Roadmap
======================

* With 'account_anglo_saxon' module is used, the price difference will be
calculated ignoring the forced exchange rate.
